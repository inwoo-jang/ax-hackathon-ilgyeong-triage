## Step 3. 코호트(프로그램)별 현재 주차

프로그램(코호트)마다 운영 진도가 달라 **현재 주차(cw)가 상이**하다. 매주 1주씩 이력이 쌓이는 구조라
"지난 주차(확정 출석률)"와 "남은 주차(만회 여지)"가 코호트별로 다르게 나뉜다.

| 현재 주차 | 인원 | 해당 코호트 |
|---|---|---|
| 2주차 | 60 | 미래내일 일경험_A · 직무훈련_C |
| 3주차 | 30 | 직무훈련_A |
| 4주차 | 30 | 미래내일 일경험_D |
| 5주차 | 60 | 미래내일 일경험_B · 직무훈련_D |
| 6주차 | 30 | 직무훈련_B |
| 7주차 | 30 | 미래내일 일경험_E |
| 8주차 | 60 | 미래내일 일경험_C · 직무훈련_E |

> 5주차 이상(P2 단계) 코호트는 P1(1~4주)이 이미 정산 확정 → **서류 미비 없음**.
> 현재 진행 중인 기간만 만회 판단 대상.

## Step 4. 출석 시간 구조 + 정산

- 두 수당(**직무훈련 `jt` / 일경험 `we`**)을 **각각 따로 계산해 합산**. (program_name은 수당종류 무관)
- 주차별 [출석 h / 배정 h]를 그대로 보유. 직무훈련 배정 10~12h, 일경험 배정 8~10h로 **주차마다 변동**.
- **정산: 4주 단위 2회차.** P1=1~4주, P2=5~8주. 각 기간 끝(4주/8주)에 정산 확정.
- 4주 구간 출석률 = `Σ출석 ÷ Σ배정`(구간 합), 직무훈련·일경험 따로.

## Step 5. 규칙 계산 (결정론 — 코드로 완결, AI 아님)

- **수당표**(브리프 값):
  - 직무훈련: 90%+ 150,000 / 70-90% 131,250 / 60-70% 112,500 / 50-60% 93,750 / <50% 0
  - 일경험: 90%+ 300,000 / 70-90% 262,500 / 60-70% 225,000 / 50-60% 187,500 / <50% 0
- **kcalc**(현재 기간 기준): cur(지난 주차 확정 출석률) / max(남은 주 다 채울 때 상한) / min(남은 주 0일 때 하한).
- **feasib(만회 판정)**: 만회불가(직무·일경험 둘 다 max<50%) / 안전(둘 다 cur≥90%) / **출석 장려 대상**(그 외).
- **개입가치(swing)** = max(0, 최대출석시 수당 − 현재 출석률 유지시 수당). cur 없으면 직전 P1 출석률로 대체. **큐 기본 정렬키.**
- **처리 대상(actionable)** = 출석 장려 대상 또는 (서류 미비 & 만회 가능).

## Step 7. 인사이트 3줄 (발표용)

1. **300명 중 229명이 "출석 장려 대상"** — 남은 출석을 채우면 수당 구간이 바뀌는 사람들. 출석률을 %가 아니라 "수당 경계까지의 여지"로 보면, 개입으로 지급액을 방어할 여력이 232명에 걸려 있다. 트리아지의 존재 이유가 데이터로 증명됨.
2. **오늘 개입으로 좌우되는 금액이 ₩16,631,250** — 운영자가 출석 장려 대상을 개입가치(swing) 큰 순으로 처리하면 이 금액이 실제 지급 방어로 이어진다. 큐 상단 = 가장 큰 돈이 걸린 사람.
3. **서류 미비 10명, 보류 금액 ₩4,050,000은 출석과 무관한 행정 블로커** — 출석이 양호해도 서류 완비 전까지 미지급. 별도 헤드라인으로 분리해 "서류 독촉" 트랙으로 처리.

## ⚠️ 발표/구현 시 주의 (함정)

1. **program_name으로 수당종류를 가르지 말 것**: `직무훈련_A` 참여자에게도 일경험 시간이, `미래내일 일경험_A`에게도 직무훈련 시간이 채워져 있음. **두 수당을 각각 계산해 합산**하는 구조.
2. **두 수당은 별도 합산**: 직무훈련(`jt`)·일경험(`we`)을 각각 출석률→구간→금액으로 산정한 뒤 더한다. 한쪽이 미지급이어도 다른 쪽은 지급될 수 있다.
3. **주차별 배정시간 변동 → 고정 분모 금지**: 배정시간(`*_wt`)이 매주 다르다. 출석률은 반드시 4주 구간의 `Σ출석 ÷ Σ배정`으로 계산. "주당 N시간" 같은 고정값으로 나누면 틀린다.
4. **이름 단독 키 금지**: `nm`은 실명이 아니며 동명이인이 있을 수 있다. 식별은 `id`(레코드 단위)로.
5. **코호트별 현재 주차 차이**: cw가 2~8주로 분산. P2 단계(5주+) 코호트는 P1 서류 완비 → 현재 진행 기간만 만회 판단.
