# 미래내일 일경험 · 운영 트리아지 어드민

> 300명 참여자의 8주 출결을 보고 **"누구에게 먼저 개입하면 수당이 지켜지는지"** 를 자동으로 띄워 주는 운영자 대시보드.
> AX 해커톤 과제3(소셜혁신연구소) · 데모

**Before → After**
운영자가 300명 × 8주 출결을 엑셀로 수기 점검(건당 3~5분 × 300명 ≈ 15~25시간/회) → **위험·개입가치 순 자동 정렬 + AI 안내문구 생성**으로 수 분.

---

## 30초 데모 시나리오

1. **오늘 할 일** — 개입가치(더 받을 수 있는 수당) 높은 순으로 대상이 정렬돼 있다.
2. 한 명 클릭 → **만회 분석**(현재 출석률 + 도달 가능 구간), **What-if 시뮬레이터**(남은 주차 출석 시간 입력 → 수당 실시간 재계산).
3. **AI 조치 카드** — 상태 요약·우선순위 근거·문자 문구·전화 스크립트 생성. "추가 지시"에 _참여자 상황·톤_ 을 적고 **다시 생성**.
4. **연락 완료 / 출석시간 수정 / 보류**(4·8주는 **확정 완료**) — 처리하면 **처리 내역**에 기록.

---

## 🤖 AI 연동 — 실서비스 vs 데모 우회 (중요)

설계 원칙: **계산·임계판정 = 규칙(코드) / 요약·생성·설명 = AI / 최종 확정 = 사람.** 계산을 AI에 떠넘기지 않는다.

| 단계 | 담당 | 내용 |
|---|---|---|
| 출석률·수당구간·금액환산·개입가치·What-if 재계산 | 🧮 **규칙(코드)** | 결정론. 정답이 있는 영역 |
| 상태 요약·우선순위 근거·문자/전화 문구 생성 | 🤖 **AI(Claude)** | 비정형 → 사람 말. 생성·설명 |
| 연락·확정·예외 판단·책임 | 🙋 **사람** | 자동 확정/발송 없음 |

### 실서비스에서는 — **AI API 키가 반드시 필요하다**
각 AI 지점은 백엔드를 통해 **Anthropic Claude API**(`POST /api/ai/recommend`)를 호출한다. **API 키는 서버에 보관**하고 프론트에 노출하지 않는다.

### 데모에서는 — 단일 HTML이라 우회했다 (3경로, 자동 폴백)
브라우저로 더블클릭해 연 단일 HTML은 키·백엔드가 없어 원래는 API를 직접 못 부른다. 그래서 아래로 우회한다. **우상단에서 모드를 고를 수 있고, 안 되면 자동으로 mock으로 떨어진다.**

| 경로 | 키 필요 | 방식 | 표시 |
|---|---|---|---|
| ① **로컬 Claude 브리지** (권장) | ❌ | `bridge.py`가 설치된 **Claude Code CLI(`claude`)** 를 호출. 기존 로그인 사용, 추가 비용 없음 | 🟢 로컬 Claude |
| ② **Anthropic API 키** 런타임 입력 | ✅ `sk-ant-…` | 브라우저에서 Claude API 직접 호출(`anthropic-dangerous-direct-browser-access` 헤더) | 🟢 실시간 AI · API |
| ③ **mock** (기본·백업) | ❌ | 규칙 기반 사전 생성 문구. 인터넷·키 없이 동작(녹화 안정) | ⚪ mock |

> ⚠️ **데모용 우회임을 명시**: ①의 로컬 브리지와 ②의 브라우저 직접 호출(`dangerous-direct-browser-access`)은 **데모/로컬 시연 전용**이다. 실제 서비스에서는 키를 서버에 두고 백엔드에서 호출해야 한다.

**환각 방지**: 어느 경로든, 규칙(코드)이 만든 **숫자·사실**(출석률·남은 주차·필요 시간·수당 구간)을 "기준 초안"으로 고정하고, AI는 그 숫자를 **바꾸지 않고 표현만** 다시 쓴다.

## 실행 방법

### 1) 그냥 보기 (mock)
`index.html` 더블클릭. AI 문구는 규칙 기반 mock으로 나온다. 키·인터넷 불필요.

### 2) 실시간 AI — 로컬 Claude 브리지 (권장, 키 불필요)
```bash
python3 bridge.py          # http://localhost:8787 (Claude Code CLI 필요)
```
→ `index.html` 열고 우상단 **로컬 Claude 연결** 클릭 → 배지가 🟢 로컬 Claude → "다시 생성"이 실제 Claude로 동작.

### 3) 실시간 AI — Anthropic API 키
`index.html` 우상단 입력칸에 `sk-ant-…` 키 입력 → 🟢 실시간 AI · API.

### 빌드 (소스 수정 시)
```bash
python3 build.py           # src/ → index.html 로 인라인
```

## 분석 모델 한눈에

- **정산 구조**: 4주 단위 2회차. P1=1~4주, P2=5~8주. 각 기간 끝(4·8주)에 정산 확정. 프로그램(멘토 그룹)별 현재 주차 상이.
- **수당**: 직무훈련·일경험 **각각** 4주 출석률 → 구간(90/70/60/50%, 50% 미만 미지급)별 차등.
- **만회 판정(feasib)**: `만회불가`(남은 주 다 채워도 미지급) / `안전`(이미 90%+ 잘하고 있음) / **`출석 장려 대상`**(아직 올릴 여지 → 처리 대상).
- **개입가치(swing)**: `최대 출석 시 수당 − 현재 출석률 유지 시 수당`. 지금 독려해 더 받을 수 있는 금액. 큐 기본 정렬 기준.
- **처리 상태**: 미처리 → 연락 완료 / 확정 완료(4·8주) / 보류.

> 데이터는 전부 **더미**다. 실명·실연락처가 아니다.
